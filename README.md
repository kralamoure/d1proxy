# d1proxy

`d1proxy` is a proxy for Dofus 1 login/game servers.

## Requirements

- [Git](https://git-scm.com/)
- [Go](https://golang.org/)

## Build

```sh
git clone https://github.com/kralamoure/d1proxy
cd d1proxy
go build ./cmd/...
```

## Installation

```sh
go get -u -v github.com/kralamoure/d1proxy/...
```

## Usage

```sh
d1proxy --help
```
